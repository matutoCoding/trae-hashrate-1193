# 口腔正畸复诊管理系统

## 系统概述

本系统是专为口腔正畸科设计的桌面客户端管理系统，实现患者按矫治阶段每周固定时段复诊的全流程管理，支持周期批量排程、单独调整、现场排队叫号、VIP/急诊优先插队等功能。

## 技术栈

- **前端框架**: Python + PySide6 (Qt for Python)
- **数据库**: SQLite (轻量级本地数据库)
- **运行环境**: Windows / macOS / Linux

## 功能模块

### 1. 复诊排期模块
- **诊疗椅建档**: 管理诊疗椅信息，支持增删改查
- **周期规则设定**: 设置每周固定时段的复诊规则
- **排期管理**: 查看和管理每日排期，支持预约/取消预约

### 2. 周期生成模块
- **按规则批量生成**: 根据周期规则批量生成未来排期
- **批量生成空时段**: 生成可预约的空时段
- **患者周期排期**: 为患者按周期批量安排复诊
- **排期列表**: 查看和编辑所有排期，支持单独调整

### 3. 排队叫号模块
- **患者取号**: 现场取号，支持关联预约
- **优先级队列**: 按患者类型自动排序队列
- **叫号功能**: 按优先级叫号，显示叫号屏
- **完成/取消**: 完成就诊或取消取号

### 4. 优先插队模块
- **VIP插队**: VIP患者可优先插队
- **急诊插队**: 急诊患者最高优先级，直接插队到最前
- **优先级规则**: 4级优先级（急诊100 > VIP80 > 优先50 > 普通0）
- **插队管理**: 查看和管理插队操作

### 5. 复诊提醒模块
- **待发送提醒**: 查看即将发送的提醒
- **提醒记录**: 查看历史提醒发送记录
- **提醒设置**: 配置提醒提前时间、消息模板
- **自动提醒**: 系统自动检查并发送提醒

## 优先级规则

| 患者类型 | 优先级 | 可插队 | 说明 |
|---------|--------|--------|------|
| 急诊 | 100 | ✅ | 最高优先级，可直接插队到最前 |
| VIP贵宾 | 80 | ✅ | 高优先级，可插队到普通患者前 |
| 复诊优先 | 50 | ❌ | 中等优先级，正常排队但优先于普通患者 |
| 普通患者 | 0 | ❌ | 正常优先级，按取号顺序排队 |

- 部分已预约的复诊

## 使用流程

### 1. 基础配置（首次使用）

1. **添加诊疗椅**: 进入"复诊排期" → "诊疗椅管理"，添加诊疗椅
2. **设置周期规则**: 进入"复诊排期" → "周期规则"，设置每周固定时段
3. **添加患者**: 在排期或排队模块中添加患者信息

### 2. 批量生成排期

1. 进入"周期生成"模块
2. 选择"周期批量生成"标签
3. 选择周期规则，设置日期范围
4. 点击"按规则生成"批量创建排期
5. 或使用"患者周期排期"为特定患者安排固定周期的复诊

### 3. 日常就诊流程

1. **患者取号**: 进入"排队叫号" → "排队叫号"标签，点击"患者取号"
2. **叫号**: 点击"叫下一位"，按优先级自动叫号
3. **显示叫号屏**: 点击"显示叫号屏"，外接显示屏可展示叫号信息
4. **完成就诊**: 患者就诊完成后点击"完成当前"

### 4. 优先插队处理

1. 进入"排队叫号" → "优先插队"标签
2. 选择可插队的患者（急诊或VIP）
3. 点击"优先插队"，选择插队位置
4. 确认后队列自动重新排序

### 5. 复诊提醒

1. 进入"复诊提醒"模块
2. 点击"批量生成提醒"生成即将到来的复诊提醒
3. 系统会自动检查并在预约时间前发送提醒
4. 可在"提醒设置"中配置提醒参数

## 特色功能

1. **智能周期生成**: 支持按周自动重复生成排期，可指定患者或生成空时段
2. **动态优先级队列**: 队列实时按优先级排序，高优先级患者自动靠前
3. **灵活插队机制**: 只有授权的患者类型（急诊、VIP）才能插队，系统自动校验
4. **可视化叫号屏**: 支持外接显示屏展示叫号信息，实时更新
5. **自动提醒系统**: 可配置提前提醒时间，自动发送复诊提醒
6. **示例数据初始化**: 一键创建测试数据，方便演示和测试

## 常见问题

**Q: 如何修改患者类型？**
A: 在取号时可以选择患者类型，或在排期模块中编辑患者信息。

**Q: 如何调整已生成的排期？**
A: 进入"周期生成" → "患者周期排期"或"排期列表"，找到要调整的排期，点击"调整"按钮。

**Q: 叫号屏如何外接显示器？**
A: 点击"显示叫号屏"按钮，将弹出的窗口拖拽到外接显示器并最大化即可。

**Q: 提醒功能如何对接短信/微信？**
A: 当前版本在控制台打印提醒信息。如需对接实际的短信或微信接口，可修改 `reminder_module.py` 中的 `_send_reminder` 方法。
